# Skills

A collection of agent skills.

## Install

```shell
npx skills add skill-tissue/skills
```

## Skills list

- [`/backseat`](./skills/backseat/SKILL.md) - Backseat gaming mode. See exactly what's wrong and tell the user precisely what to do, but never touch the code. The user implements everything.
- [`/cvmn`](./skills/cvmn/SKILL.md) - Caveman mode plus vowel-dropping. Drops articles, fillers, pleasantries, hedging, AND drops all vowels from remaining prose — code/identifiers/quoted errors stay intact.
